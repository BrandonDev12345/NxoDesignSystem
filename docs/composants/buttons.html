Comportement responsive

Les boutons s'adaptent automatiquement sur mobile. Les groupes de boutons (.nxo-btn-group) passent en retour a la ligne puis en colonne sur petit ecran.
Breakpoint Comportement
≤ 768px .nxo-btn-group passe en flex-wrap: wrap — les boutons retournent a la ligne si necessaire
≤ 480px .nxo-btn-group passe en flex-direction: column — boutons empiles en pleine largeur

Voir la page Responsive pour tous les details.
