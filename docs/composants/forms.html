  <input type="text" class="nxo-input nxo-input-success" value="Valeur correcte">

  </span>

<div class="nxo-form-group">
  <label class="nxo-label">Champ en erreur</label>

Comportement responsive

Les champs de formulaire s'adaptent automatiquement. Sur petit mobile, les inputs et selects sont agrandis pour eviter le zoom automatique sur iOS.
Breakpoint Comportement
≤ 480px Les .nxo-input et .nxo-select passent a height: 44px et font-size: 16px pour eviter le zoom iOS
