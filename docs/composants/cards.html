Cards

radius: 8px

Conteneurs flexibles pour regrouper du contenu. Utilisez les cards pour des articles, statistiques, ou actions.

Card basique

Preview

Titre de la card

Description courte du contenu de cette card. Peut contenir plusieurs lignes de texte.

<div class="nxo-card">
  <div class="nxo-card-body">
    <h4 class="nxo-card-title">Titre de la card</h4>
    <p class="nxo-text-sm nxo-text-muted">Description...</p>
    <button class="nxo-btn nxo-btn-primary nxo-btn-sm">Action</button>
  </div>
</div>
// ⚠️ Exemple avec Material UI + classes NXO

import { Card, CardContent, Button, Typography } from '@mui/material';

<Card className="nxo-card">
  <CardContent className="nxo-card-body">
    <Typography variant="h6" className="nxo-card-title">
      Titre de la card
    </Typography>
    <Typography className="nxo-text-sm nxo-text-muted">
      Description...
    </Typography>
    <Button className="nxo-btn nxo-btn-primary nxo-btn-sm">
      Action
    </Button>
  </CardContent>
</Card>
// ⚠️ Exemple avec Angular Material + classes NXO

import { MatCardModule } from '@angular/material/card';
import { MatButtonModule } from '@angular/material/button';

// Dans le template
<mat-card class="nxo-card">
  <mat-card-content class="nxo-card-body">
    <h4 class="nxo-card-title">Titre de la card</h4>
    <p class="nxo-text-sm nxo-text-muted">Description...</p>
    <button mat-button class="nxo-btn nxo-btn-primary nxo-btn-sm">
      Action
    </button>
  </mat-card-content>
</mat-card>

// ⚠️ Exemple avec Material UI + classes NXO

import { Card, CardContent, CardActionArea, Typography } from '@mui/material';
import { FontAwesomeIcon } from '@fortawesome/react-fontawesome';
import { faFolder, faChevronRight } from '@fortawesome/free-solid-svg-icons';
import { useNavigate } from 'react-router-dom';

const navigate = useNavigate();

<Card className="nxo-card nxo-card-hover">
  <CardActionArea onClick={() => navigate('/documents')}>
    <CardContent className="nxo-card-body">
      <div style={{ display: 'flex', alignItems: 'center', gap: 12 }}>
        <div className="nxo-card-icon">
          <FontAwesomeIcon icon={faFolder} />
        </div>
        <div>
          <Typography className="nxo-card-title">Documents</Typography>
          <Typography className="nxo-text-sm nxo-text-muted">23 fichiers</Typography>
        </div>
        <FontAwesomeIcon icon={faChevronRight} style={{ marginLeft: 'auto' }} />
      </div>
    </CardContent>
  </CardActionArea>
</Card>

Comportement responsive

Les cards s'adaptent en reduisant leur padding interne sur mobile pour maximiser l'espace de contenu.
Breakpoint Comportement
≤ 768px Padding du header, body et footer reduit (spacing-sm / spacing-md)
≤ 480px Padding du body reduit a spacing-sm

Voir la page Responsive pour tous les details.
