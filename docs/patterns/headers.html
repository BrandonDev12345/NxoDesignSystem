App Launcher (Pave 9 services)

Le pave a droite du header (.nxo-header__icon--apps) permet au client de naviguer entre les differents services en ligne NXO. Le menu burger est toujours visible a gauche sur toutes les tailles. Sur tablette/mobile (≤ 1024px), le pave 9 dots est masque et les services sont accessibles via le menu burger lateral.

Icone 9 points : L'icone grille 3x3 n'existe pas dans Font Awesome. C'est une icone CSS custom (.nxo-header__grid-icon) composee de 9 <span> dans une grille CSS. Voir la section "Classes CSS" ci-dessous pour le detail.

<!-- JavaScript -->
<script>
function toggleAppLauncher() {
  document.getElementById('appLauncher').classList.toggle('open');
}
document.addEventListener('click', function(e) {
  var btn = document.querySelector('.nxo-header__icon--apps');
  var panel = document.getElementById('appLauncher');
  if (btn && panel && !btn.contains(e.target) && !panel.contains(e.target)) {
    panel.classList.remove('open');
  }
});
</script>

Menu Mobile (Burger)

Le bouton hamburger est toujours visible a gauche du header sur toutes les tailles. Il ouvre un menu lateral (sidebar) qui slide depuis la gauche et contient :

  • Navigation du service — liens internes de l'application (Accueil, Demandes, Rapports...)
  • Services NXO — les 9 services en liste verticale avec icone + label (memes liens que l'App Launcher)
Le menu mobile est commun a tous les breakpoints ≤ 1024px (tablet, tablet small, mobile). La navigation du service est personnalisable, mais la section "Services NXO" reste identique partout.

Code HTML

<div class="nxo-header__mobile-overlay" id="mobileOverlay"></div>

<!-- Menu lateral -->

      <i class="fa-solid fa-xmark"></i>
    </button>

    <!-- Navigation du service -->

        Espace client

    </div>

</nav>
