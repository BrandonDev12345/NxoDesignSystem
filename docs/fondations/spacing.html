Espacements

Echelle d'espacements basee sur une unite de 4px pour garantir la coherence visuelle.

Echelle d'espacements

VariableValeurUsageVisualisation
--nxo-spacing-xs 4px Entre icone et texte
--nxo-spacing-sm 8px Elements proches
--nxo-spacing-md 16px Padding standard
--nxo-spacing-lg 24px Entre sections
--nxo-spacing-xl 32px Grands espaces
--nxo-spacing-2xl 48px Blocs majeurs

Classes utilitaires

ClassePropriete
.nxo-mb-xs margin-bottom: 4px
.nxo-mb-sm margin-bottom: 8px
.nxo-mb-md margin-bottom: 16px
.nxo-mb-lg margin-bottom: 24px
.nxo-mb-xl margin-bottom: 32px

Usage en CSS

/* Utiliser les variables d'espacement */
.mon-element {
  padding: var(--nxo-spacing-md);
  margin-bottom: var(--nxo-spacing-lg);
  gap: var(--nxo-spacing-sm);
}

/* Ou avec les classes utilitaires */
<div class="nxo-mb-lg">...</div>
