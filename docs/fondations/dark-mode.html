Dark Mode

Le dark mode est integre au theme NXO. Tous les composants basculent automatiquement via l'attribut data-theme="dark" sur la balise <html>. Aucun fichier CSS supplementaire n'est necessaire.

Comment ca marche

Le dark mode repose sur le mecanisme CSS natif de surcharge de variables. Quand data-theme="dark" est present sur <html>, les tokens definis dans tokens.css sont remplaces par leurs equivalents sombres. Les composants de components.css utilisent ces memes variables, donc ils s'adaptent automatiquement.

Mise en place

1
Imports CSS (inchanges)
Les memes deux fichiers que d'habitude. Le dark mode est inclus dedans.
<link rel="stylesheet" href="tokens/tokens.css">
<link rel="stylesheet" href="components/components.css">
2
Script anti-flash (dans le <head>)
Place ce script avant les liens CSS pour eviter le flash blanc au chargement quand l'utilisateur a choisi le dark mode. 
<script>
(function() {
  var theme = localStorage.getItem('nxo-theme');
  if (theme === 'dark' || (!theme &&
      window.matchMedia('(prefers-color-scheme: dark)').matches)) {
    document.documentElement.setAttribute('data-theme', 'dark');
  }
})();
</script>
3
Fonction de toggle
Ajoute cette fonction et lie-la a un bouton. Elle bascule le theme et sauvegarde le choix dans localStorage. 
function toggleDarkMode() {
  const html = document.documentElement;
  if (html.getAttribute('data-theme') === 'dark') {
    html.removeAttribute('data-theme');
    localStorage.setItem('nxo-theme', 'light');
  } else {
    html.setAttribute('data-theme', 'dark');
    localStorage.setItem('nxo-theme', 'dark');
  }
}

// Exemple de bouton
<button onclick="toggleDarkMode()">
  Changer de theme
</button>
4
Ecouter le changement systeme (optionnel)
Si l'utilisateur n'a pas fait de choix explicite, le theme suit la preference systeme (Windows/macOS/navigateur). 
window.matchMedia('(prefers-color-scheme: dark)')
  .addEventListener('change', function(e) {
    if (!localStorage.getItem('nxo-theme')) {
      if (e.matches) {
        document.documentElement.setAttribute('data-theme', 'dark');
      } else {
        document.documentElement.removeAttribute('data-theme');
      }
    }
  });
