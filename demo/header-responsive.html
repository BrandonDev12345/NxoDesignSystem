Header NXO Responsive

Redimensionne la fenetre de ton navigateur pour voir le header s'adapter automatiquement.

Breakpoints :

  • > 1440px : Desktop XL - Grip + Logo + Nom + Search bar + 3 icones (80px)
  • 1024px - 1440px : Desktop - Meme layout, tailles reduites (80px)
  • 768px - 1024px : Tablet - Hamburger + Logo + Nom + Search bar + 3 icones (72px)
  • 480px - 768px : Tablet Small - Hamburger + Logo + 4 icones (64px)
  • < 480px : Mobile - Hamburger + Logo centre + 2 icones (56px)

Comment utiliser

Importe tokens.css et components.css dans ton projet. Toutes les classes .nxo-header__* sont disponibles dans le theme.
